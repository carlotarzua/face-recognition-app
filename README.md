🧠 Face Recognition App

A real-time face recognition system built with Python, OpenCV, and the `face_recognition` library.  
It detects and identifies faces from a webcam feed using known images.

📸 Features

- Load and encode known faces from a dataset folder
- Real-time webcam feed with face detection
- Name labels over recognized faces
- Easy to extend with new people and datasets

⚠️ The face dataset is not included in this repo due to size limit
